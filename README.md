# Ancient Town Kusabuki
Kusabuki is a fictitious town surrounded by mountains on three sides with the east side facing the sea.

## Install
```
yarn
```

## Usage
```
yarn start
```

## Development Tools
- Nuxt.js (using Vuex)

## Design
- Responsive Web Design
- Original loading animation
